# Privacy Policy - MineHostLocal

**Last updated: 2026-06-30**

## Data Collection

MineHostLocal stores all user data **locally on your device only**:
- Username and password (hashed, local only)
- Server configurations and settings
- Backup files

## Tunnel Service (minehost-local.uk)

When using the optional tunnel feature, the following is processed:
- **IP address**: Used temporarily for routing connections to your server
- **Server name**: Used as a subdomain (e.g. myserver.minehost-local.uk)

This data is **not sold or shared** with third parties and is only used to provide the tunnel functionality.

## No Analytics

MineHostLocal does not collect analytics, crash reports, or usage statistics.
